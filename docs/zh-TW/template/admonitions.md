# 警告、提示等

警示（Admonitions），也稱為「標註（Call-outs）」，是一種很好的方式，可以在不大幅影響文件主線的情況下加入補充內容。

欲了解更多詳細資訊，請參閱官方文件：[Admonitions 詳細說明](https://squidfunk.github.io/mkdocs-material/reference/admonitions/)

## 可折疊區塊

當 Details 功能啟用時，如果警示區塊以 ??? 而非 !!! 開頭，則該警示會呈現為可展開的區塊，並在右側顯示一個小型切換按鈕
